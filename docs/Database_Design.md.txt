# DevFlow AI - Database Design

## Tables
- users
- profiles
- projects
- project_notes
- project_links
- skills
- user_skills
- interview_questions
- coding_logs

## Relationships
- One user has one profile.
- One user has many projects.
- One project has many notes.
- One project has many links.
- One user can have many skills.
- One user can have many interview questions.
- One user can have many coding logs.
